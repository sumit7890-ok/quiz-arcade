# 🏆 Quiz Arcade

A modern retro-themed quiz application built using HTML, CSS, and JavaScript.

Test your knowledge, challenge yourself, and track your performance through an interactive and visually appealing quiz experience.

---

## 📸 Preview

### Start Screen
- Retro pixel-style title
- Difficulty selection (Easy, Medium, Hard)
- Floating animated question marks
- Glassmorphism-inspired background
- Dynamic motivational quotes

### Quiz Screen
- Question palette navigation
- Timer countdown
- Previous / Next navigation
- Real-time question status tracking

### Result Screen
- Score summary
- Answer review
- Correct and incorrect answer analysis

## 🎮 How To Play

1. Select a difficulty level.
2. Click Start Quiz.
3. Answer the questions before time runs out.
4. Navigate using question numbers or Next/Previous buttons.
5. Submit the quiz.
6. Review your score and answers.
